    • 分析:

    测试数据uer对象中userName值为null, 并且没有指定jdbcType时会出现异常, 文档http://mybatis.github.io/mybatis-3/sqlmap-xml.html 中指明:

    NOTE The JDBC Type is required by JDBC for all nullable columns, if null is passed as a value. You can investigate this yourself by reading the JavaDocs for thePreparedStatement.setNull() method.

    • 解决方案:

    每个需要设置值的字段都指定jdbcType

    @Insert("insert into sys_user(user_id, user_Name, PASSWORD, UPDATED_WHEN) values(#{userId, jdbcType=VARCHAR},#{userName, jdbcType=VARCHAR},#{password, jdbcType=VARCHAR},#{updatedWhen, jdbcType=DATE})")
public void insert(User user);
