    Vuepress文章刷新后页面异常

    Hyman 2020/11/8 实践总结Vue

    今天用Vuepress搭建了博客，搭建步骤：

    1. 安装Nginx，以为觉得是静态文件发布，就没有修改默认配置。
    2. Vuepress编辑后，将生成的public里的内容直接拷贝到**/usr/share/nginx/html**

    拷贝完成后，直接访问博客就能正常访问，但是出现了个奇怪现象，在首页点击文章都能正常，但是进入文章后刷新就出现：

    猜想应该是nginx的location配置有问题，上网一查，原来还需要加上location里还需要配置一个try_files $uri $uri/ /index.html;

    如下：

    location / {
  try_files $uri $uri/ /index.html;
  index index.html index.htm;
}

    配置完service nginx reload，刷新正常
